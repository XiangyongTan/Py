plt.plot(x,t,format_string,kwargs)

  • format_string 控制曲线的格式字符串,由颜色字符，风格字符和标记字符组成

  • kwargs中的设置也可以在format_string设置，效果都一样。

pyplot显示中文的第二种方法，即在有中文出现的地方增加一个属性，fontproperties,而第一种方法改变了全局的效果，比如横轴上的数字也变大了，但这一种只改变你想改变的地方，其他不变，各有利弊吧。

plt.show()

pyplot中文本显示函数¶

  • plt.xlabel():对x轴增加文本标签
  • plt.ylabel():对y轴增加文本标签
  • plt.title():对图形整体增加文本标签
  • plt.text():对任意位置增加文本
  • plt.annotate():在图形中增加带箭头的注释

pyplot的基础函数¶

  • plt.plot()
  • plt.boxplot()：箱线图
  • plt.bar():条形图
  • plt.barh()：横向条形图
  • plt.polar()：极坐标图
  • plt.pie()：饼图
  • plt.psd():绘制功率谱密度图
  • plt.specgram():绘制谱图
  • plt.choere(x,y):绘制相关性图
  • plt.scatter():散点图
  • plt.step()：绘制阶梯图
  • plt.hist():直方图
  • plt.contour():等高图
  • plt.vlines():垂直图
  • plt.stem():柴火图
  • plt.plot_data():数据日期

import numpy as np
import matplotlib.pyplot as plt
from scipy.io import wavfile
 
rate_h, hstrain= wavfile.read(r"H1_Strain.wav","rb")
rate_l, lstrain= wavfile.read(r"L1_Strain.wav","rb")
#reftime, ref_H1 = np.genfromtxt('GW150914_4_NR_waveform_template.txt').transpose()
reftime, ref_H1 = np.genfromtxt('wf_template.txt').transpose() #使用python123.io下载文件
 
htime_interval = 1/rate_h
ltime_interval = 1/rate_l
fig = plt.figure(figsize=(12, 6))
 
# 丢失信号起始点
htime_len = hstrain.shape[0]/rate_h
htime = np.arange(-htime_len/2, htime_len/2 , htime_interval)
plth = fig.add_subplot(221)
plth.plot(htime, hstrain, 'y')
plth.set_xlabel('Time (seconds)')
plth.set_ylabel('H1 Strain')
plth.set_title('H1 Strain')
 
ltime_len = lstrain.shape[0]/rate_l
ltime = np.arange(-ltime_len/2, ltime_len/2 , ltime_interval)
pltl = fig.add_subplot(222)
pltl.plot(ltime, lstrain, 'g')
pltl.set_xlabel('Time (seconds)')
pltl.set_ylabel('L1 Strain')
pltl.set_title('L1 Strain')
 
pltref = fig.add_subplot(212)
pltref.plot(reftime, ref_H1)
pltref.set_xlabel('Time (seconds)')
pltref.set_ylabel('Template Strain')
pltref.set_title('Template')
fig.tight_layout()
 
#plt.savefig("Gravitational_Waves_Original.png")
plt.show()
plt.close(fig)
