URL的一般格式为(带方括号[]的为可选项)¶

  • protocol://hostname[:port]/path/[:parameter][?query]
  • 第一部分是协议:http,https,ftp,file...
  • 第二部分是存放资源的服务器的域名系统或IP地址(有时候要包括端口号，各种传输协议都有默认的端口号，如http的默认端口为80)
  • 第三部分是资源的具体地址，如目录或文件名等。

步骤¶

  • proxy_support=urllib.request.ProxyHandled({})参数是一个字典{"类型"："赛丽IP:端口号"}.
  • 定制、创建一个opener,opener=urllib.request.build_opener(proxy_support)
  • 安装 opener,有以下两种方式 
     (1) urllib.request.install_opener(opener)
 (2) 调用opener，即opener.open(url)

def find_imas(url):
    html = url_open(url).decode("utf-8")
    img_addrs = []
    a = html.find("img src=")
    
    while a!= -1:
        b = html.find(".jpg",a,a+255)
        if b!=-1:
            img_addrs.append(html[a+9:b+4])
        else:
            b=a+9
        a=html.find("img src=",b)
        
    return img_addrs

def get_img(html):
    p=r'img src="([^"]+\.jpg)"'
    imglist=re.findall(p,html)#获取图片的地址如此简单
   
    #for each in imglist:

Scrapy¶

  • Scrapy 是为了爬取网站数据，提取结构性数据而编写的框架。可以应用在包括数据挖掘，信息处理或存储历史数据等一系列的程序组。
  • Scrapy 最初是为了页面抓取（更确切的说是 网络抓取）所设计的，也可以应用在获取API所返回的数据或者通用的爬虫。
