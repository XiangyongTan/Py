Requests库的7个主要方法¶

  • requests.request(): 构造一个请求，支撑以下各方法的基础方法;
  • requests.get(): 获取HTML网页的主要方法，对应于HTTP的GET;
  • requests.head(): 获取HTML网页头信息方法，对应于HTTP的HEAD;
  • requests.post(): 向HTML网页提交POST请求的方法，对应于HTTP的POST;
  • requests.patch(): 向HTML网页提交局部修改请求，对应于HTTP的PATCH;
  • requests.delete(): 向HTML网页提交删除请求，对应于HTTP的DELETE;

使用requests.get()返回有以下属性¶

  • r.status_code: HTTP请求返回的状态，200表示成功连接，404表示失败；
  • r.text: HTTP响应内容的字符串形式，即url对对应页面的内容；
  • r.encoding: 从HTTP header中猜测的响应内容编码方式；
  • r.apparent_encoding: 从内容中分析出的响应内容编码方式；
  • r.content: HTTP响应内容的二进制形式；

理解Rquests库的异常¶

  • request.ConnectionError: 网络简介错误异常，入DNS查询失败，拒绝连接等；
  • requests.HTTPError: HTTP错误异常；
  • requests.URLRequired: URL缺失异；
  • requests.TooManyRedirects: 超过最大重定向次数，产生重定向异常；
  • requests.ConnectTimeout: 连接远程服务器超时异常；
  • requests.Timeout: 请求URL超时，产生超时异常；
  • r.raise_for_status(): 如果不是200，产生requests.HTTPError异常

HTTP¶

  • HTTP: Hypertext Transfer Protocal,超文本传输协议；
  • HTTP是一个基于“请求与响应”模式的，无状态是应用层协议；
  • HTTP协议采用URL作为定位网络资源的标识。

URL格式: http : // host [:port] [path]¶

  • host: 合法的Internet主机域名或者IP地址。
  • port: 端口号， 如http的默认端口为80。
  • path: 资源的具体地址，如目录或文件名等。

HTTPURL的理解：URL是通过HTTP协议存储资源的Internet路径。一个URL对应一个数据资源。¶

HTTP协议对资源的操作¶

  • GET： 请求回去URL位置资源；
  • HEAD： 请求获取URL位置资源的响应消息报告，即获取该资源的头部信息；
  • POST: 请求向URL未知的资源后附加新的数据；
  • PUT: 请求向URL未知存储一个资源，覆盖原URL未知的资源；
  • PATCH： 请求局部更新YRL未知的资源，即更改该处资源的部分内容；
  • DELETE: 请求删除URL未知存储的资源。

requests.request(method,url,**)¶
  • method:请求方式，对应get,put等7种；
  • URL： 拟获取页面的url链接；
  • **：其他控制参数；
  • 如果method=get,则requests.request(get,url,) = requests.get(url,)
  • 针对爬虫，通常，我们使用get,head。

网站为防止爬虫，会有以下两个方式：¶

  • 检查User-Agent, 即查看是否是人为还是机器发出的请求，为防止爬虫被识别出来，我们可以对User-Agent进行修改。
  • Robots: 即公告。网站告知哪些数据可以爬取，哪些不能爬取

信息标记与提取¶

信息标记的三种形式：¶

  • XML: 最早的通用信息标记语言，可扩展性好，但繁琐；
  • JSON： 信息又类型，适合程序处理，较XML简洁；
  • YAML： 信息无类型，文本信息比例较高，可读性较好；

信息提取的一般方法¶

  • 完整解析信息的标记形式，再提取关键信息；
  • 无视标记信息，直接搜素关键信息；

def fillUniList(ulist,html):
    soup = BeautifulSoup(html, "html.parser")
    ## 因为所有信息都在tbody中，一个tr对于一所学校的信息，td是学校的具体信息
    for tr in soup.find("tbody").children:
        if isinstance(tr, bs4.element.Tag):
            tds = tr("td")
            ulist.append([tds[0].string,tds[1].string,tds[2].string,tds[3].string,tds[4].string])

def parasePage(ilt,html):
    try:
        plt = re.findall(r'p-price=\"[\d\.]*\"',html)
        tlt = re.findall(r'p-name=\".*?\"',html)
        for i in range(len(plt)):
            price = eval(plt[i].split('=')[1])
            title = eval(tlt[i].split('=')[1])
            ilt.append([price,title])
    except:
        print("")

import scrapy
class DemoSpider(scrapy.Spider):
    name = 'demo'
    start_urls = ['http://python123.io/ws/demo.html']

    def parse(self, response):
        fname = response.url.split("/")[-1]
        with open(fname,"wb") as f:
            f.write(response.body)
        self.log("Save file %s." % fname)
